# My Portfolio App

Flutter portfolio application showcasing my projects, skills, and contact information.

## Features
- Animated UI
- Flip project cards
- Skills progress bars
- Contact links (GitHub, LinkedIn, WhatsApp)
- Video meeting integration

## Technologies
- Flutter
- Dart
- Material UI
- WebView
- URL Launcher

## Screens
- Home
- About Me
- My Work
- Information
- Contact
